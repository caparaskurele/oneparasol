## â“ Frequently Asked Questions

**Q: Can students edit other students' work?**
A: No. Students can only edit their own submissions.

**Q: Can I delete a single file from a submission without deleting the whole note?**
A: Yes! Expand the submission and click the trash icon next to the file.

**Q: Are there file size limits?**
A: Currently no, but you can add limits by editing StudentNoteForm.tsx.

**Q: Can I export all submissions?**
A: Not yet, but all data is in `./data/python-notes/` as JSON files.

**Q: What happens if a student uploads a large video?**
A: It will be stored and available for download. Consider setting a file size limit.

**Q: Can I backup student submissions?**
A: Yes! Simply backup the `./data/python-notes/` directory.

## ğŸš¨ Troubleshooting

### Files Won't Upload
- Check file size
- Verify browser supports the file type
- Clear browser cache and retry
- Check browser console for errors

### Submission Doesn't Appear
- Refresh the page
- Clear browser cache
- Check network tab in DevTools for errors
- Verify the API responded with 201 status

### Can't Edit My Submission
- Make sure you're clicking the âœï¸ pencil icon
- You can only edit your own submissions

- Follow the setup guide in PYTHON-LEARNING-HUB-DOCS.md to enable
- Requires authentication setup
